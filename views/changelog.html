FaeriaDecks - Changelog

Changelog

v3.7 (3/2/2014)
  • Switched to using gamepedia wiki as data source.
  • Updated card data and images.
  • Added support for Rage and Faerialink abilities.
  • Fixed render bug where cards with abilities with a value were not including the value in the background-color-highlight.
  • Removed uncommon rarity and added basic rarity.
v3.6 (12/7/2013)
  • Updated deckbuilder import/export to support both in-game and forum formats without needing to specify the format.
  • Updated cards.json to be served by the server.
  • Added jsonp for support for cards.json via: 
    http://www.faeriadecks.com/cards.json?callback=callbackFunctionName
    with jQuery 
    $.getJSON("http://www.faeriadecks.com/cards.json?callback=?",function(obj) {
	console.log( obj)
});
  • Updated images.
  • Updated faeria card render library to support multicolored cards and adjusted land icons to overlap textbox to match in-game.
v3.5 (11/30/2013)
  • Updated deckbuilder import and export to support both in-game and forum formats.
v3.4 (11/10/2013)
  • Added import and export tool. For formatting details see the forum post.
v3.3 (11/7/2013)
  • Updated card list.
v3.2 (10/20/2013)
  • Added card images to the deckbuilder.
  • Added the visual card list.
  • Updated the faeriarender lib to use new queue rendering.
  • Fixed data re-loading when navigating between pages.
  • Fixed the filters so 'E' correctly shows as 'Exceptional'.
v3.1 (10/17/2013)
  • Updated fonts on the card designer.
  • IM Fell for name and type. Prata for power/life. 16px Serif for costs. Arial for textbox.
  • Added black stroke around power/life
  • Added line break support on card designer.
v3.0 (10/15/2013)
  • Updated the cardlist.
  • Switched to using card ids from within the game. Decklists should no longer break between versions.
  • Added card names from the in-game card list that the wiki has not yet added.
  • Corrected a number of display and parsing issues with life/power.
  • Modified structures to only show life (match in-game deckbuilder).
  • Cardlist json
v2.0 (10/12/2013)
  • Updated the cardlist.
  • Added versioning for card lists. Upgrading old decks will incorrectly parse them. Until we have a full cardlist, this is an unfortunate side effect.
  • Added Nav dropdown button.
  • Added changelog page.
  • Updated faeriacards lib.
  • Cardlist json
